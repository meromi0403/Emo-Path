# 🌱 emo-path

## 💡 소개

emo-path는 사용자의 감정을 분석하고,
감정 흐름을 기반으로 공감과 행동을 제안하는 정서 케어 AI 서비스입니다.

---

## 🧠 주요 기능

* 감정 분석 (Emotion Detection)
* 공감형 AI 응답 (Empathetic Response)
* 행동 추천 시스템 (Action Recommendation)
* 감정 기록 및 시각화 (Emotion Tracking)
* 감정 흐름 기반 개인화 (Personalization)

---

## 🚀 차별점

* 단순 챗봇이 아닌 **감정 흐름 기반 AI**
* 사용자 감정 데이터를 기반으로 지속적인 케어 제공
* 향후 로봇과 연동 가능한 구조
